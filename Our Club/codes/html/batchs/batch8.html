ABSTRACT

Multi-person tracking is affected by varies parameters such as crowd density, obstacle, range, etc. The LoRa technology is new but could provide a good range and efficiency for development of tracking system. Using LoRa technology, tracking and monitoring of individuals as well as selected users using their GPS location is done using their official assets. The collected data are used for further analysis with time stamp. Thus, this system is able to track and monitor the location of the person with real time accuracy.
